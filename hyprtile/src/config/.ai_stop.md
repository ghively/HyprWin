# AI Agent Guidance — src/config/

## Purpose
TOML configuration loading, validation, and hot-reload.

## Key Types
- `Config` — Root config struct with serde.
- `ConfigManager` — Loads config, watches file, validates.
- `GeneralConfig`, `GapsConfig`, `WorkspacesConfig`, `WindowRule`, `MonitorConfig`.

## Before Modifying
- New fields need `#[serde(default = "...")]` for backward compatibility.
- Add default function in types.rs, add to default_config() in defaults.rs.
- Update docs/CONFIGURATION.md with the new option.
- Validation rules go in ConfigManager::validate().

## Extension Points
- Add new config sections as new structs.
- Add new window rule match fields (e.g., match_title_regex).
