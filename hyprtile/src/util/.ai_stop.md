# AI Agent Guidance — src/util/

## Purpose
Low-level utilities: rectangle math, DPI scaling, animation easing. No Win32 dependencies.

## Key Types
- `Rect` — Screen-space rectangle with i32 coordinates.
- `Point` — 2D point with distance calculation.
- `Easing` / `Animation` — Time-based interpolation.

## Before Modifying
- All functions must be pure (no side effects).
- Rect operations must never produce zero/negative dimensions.
- These are hot-path functions — keep them fast and simple.

## Extension Points
- Add new easing variants to `Easing` enum.
- Add new rect operations (e.g., union, scale) as methods.
