# AI Agent Guidance — src/ (Root Source)

## What This Folder Contains
The entire Rust source tree for HyprTile. Every module is `pub` and exposed through `lib.rs`.

## Before Modifying Any File Here
1. Read the corresponding `.ai_stop.md` in the subfolder.
2. Check `docs/DEVELOPERS_GUIDE.md` for architecture context.
3. Understand the module's public API surface in its `mod.rs`.

## Common Extension Patterns
- **New layout**: Add file in `layout/`, update `LayoutType`, add tests.
- **New IPC command**: Add variant in `ipc/protocol.rs`, handler in `ipc/commands.rs`.
- **New hotkey**: Add to config defaults, handler in `app.rs::handle_hotkey()`.
- **New window filter**: Add check in `window/filter.rs`, update `passes_all_filters()`.

## Critical Invariants
- Never orphan threads (always join on shutdown).
- Never panic in event handlers (use graceful fallbacks).
- Always check `IsWindow()` before Win32 API calls on foreign HWNDs.
- Keep `WindowId` as the single identity type — don't use raw `isize` or `HWND`.
